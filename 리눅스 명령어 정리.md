# 리눅스 명령어 정리
#### 2026.01.23 작성
<br>

1. `cd [디렉토리]` : 디렉토리로 이동 <br> <br>

2. `ls` : 디렉토리 목록 확인 
    - -a : 숨겨진 파일이나 디렉토리까지 보여줌
    - -l : 파일, 디렉토리 리스트와 파일의 종류 및 권한, 링크수, 사용자/그룹, 파일크기, 수정시간, 파일 이름등을 함께 출력해줌 <br>
    `ls -al` : 숨겨진 파일이나 디렉토리까지 확인 가능 <br> <br>

3. `pwd` : 현재 작업중인 디렉토리 정보 출력(절대경로 확인)<br> <br>

4. `cp [복사하고 싶은 파일][새롭게 생성할 파일]` : 파일 또는 디렉토리 복사 
   *디렉토리를 복사하는 경우 반드시 옵션을 붙여야 한다!* <br> <br>

5. `rm [파일명]` : 파일 삭제
    `rm -rf` : 강제로 하위 파일까지 모두 삭제 <br> <br>

6. `mv [옵션][파일명][이동시킬 위치]` : 파일 이동 및 파일 이름 변경<br> <br>

7. `mkdir [디렉토리 이름]` : 디렉토리 생성 <br> <br>

8. `rmdir [디렉토리 이름]` : 디렉토리 삭제 <br> <br>

9. `cat [파일명]` : 파일 내용 출력(read) <br> <br>

10. `file [옵션][파일명]` : 파일 종류 확인 <br> <br>

11. `find [OPTION][PATH][EXPRESSION]` : 특정 파일 및 디렉토리 찾기(내용X, 파일명/디렉토리명O)
    - Option
    1. **-P** : 심볼릭 링크를 따라가지 않고 심볼릭 링크 자체의 정보를 사용
    2. **-L** : 심볼릭 링크에 연결된 파일 정보를 사용
    3. **-H** : 심볼릭 링크를 따라가지 않음(명령행 인자(Command Line Argument)를 처리할 때는 예외)
    4. **-D** : 디버그 메시지 출력 <br><br>
    - Expression
    1. **-name** : 지정된 문자열 패턴에 해당하는 파일 검색
    2. **-empty** : 빈 디렉토리 / 크기가 0인 파일 검색
    3. **-path** : 지정된 문자열 패턴에 해당하는 경로에서 검색
    4. **-size** : 파일 크기를 이용해 검색
    5. **-type** : 지정된 파일 타입에 해당하는 파일 검색
    6. **-user** : 사용자가 누구인지를 이용하여 파일 검색
    7. **-group** : 그룹이 누구인지를 이용하여 파일 검색
    8. **-readable** : 읽을 권한 유무를 이용하여 파일 검색
    9. **-executable** : 실행 권한 유무를 이용하여 파일 검색
    10. **-atime** : 파일 접근 시각을 기준으로 파일 검색
    11. **-ctime** : 파일 내용 및 속성 변경 시각을 기준으로 파일 검색
    12. **-mtime** : 파일 데이터 수정 시각을 기준으로 파일 검색
    <br> <br>
    
    1. **-delete** : 검색된 파일 또는 디렉토리 삭제
    2. **-exec** : 검색된 파일에 대해 지정된 명령 실행
    3. **-print** : 검색 결과 출력. 검색 항목은 newline으로 구분
    4. **-print0** : 검색 결과를 출력. 검색 항목은 null로 구분
    5. **-mindepth** : 검색을 시작할 하위 디렉토리 최소 깊이 지정
    6. **-maxdepth** : 검색할 하위 디렉토리의 최대 깊이 지정 <br> <br>

12. `sort [OPTION][FILE]` : 파일이나 명령어 출력 결과를 사용자 정의 기준에 따라 정렬.
    - Option
    1. **-n** : 숫자 기준으로 정렬
    2. **-r** : 역순으로 정렬
    3. **-k** : 특정 필드를 기준으로 정렬(띄어쓰기 기준)
    4. **-t** : 특정 구분자를 기준으로 필드 구분 및 정렬
    5. **-u** : 중복된 라인 제거 <br> <br>

13. `uniq [OPTION]` : 이어진 내용의 중복을 제거
    - Option
    1. **-c** : 몇 번 중복되었는지 출력

`strings` : 파일에서 문자열을 추출(파일 안에 포함된 문자열 중 사람이 인식 가능한 문자열이 있는지에 대한 정보 수집)

`base64 [OPTION][FILE]` : 문자열을 base64로 언코드 또는 디코드 
    - Option
    1. **-d** : base64 디코드 (옵션이 없으면 디코드)
    2. **-i** : 디코딩 할 때 알파벳 아닌 문자는 무시
    3. **-w** : COLS 문자 뒤에 줄 바꿈

`du [OPTION][FILE]` : 디렉토리 별 디스크 사용량 알려줌
    - Option
    1. **-a** : 디렉토리뿐만 아니라 파일까지 디스크 사용량을 알려줌
    2. **-b** : 기본 단위는 1KB, 이 옵션 사용시 1b 단위로 출력
`vi` : 문서 편집기 실행
    - `i` : 편집
    - `wq` : 저장 후 종료
    - `q!` : 강제 종료
    - `w!` : 강제 저장
`head` : 파일의 앞부분을 확인
`tail` : 파일의 뒷부분을 확인
     - Option
     1. **-n** [NUMBER] : num만큼 출력
     2. **-c** [NUMBER] : num byte만큼 출력
     3. **-q** : 여러 개의 파일을 출력할 때 제목 출력 X
     4. **-f** : 내용이 변경될 때마다 실시간으로 출력
`which [명령어]` : 특정 명령어의 위치를 찾아주는 명령어
`whereis [명령어]` : 명령어의 실행파일 위치, 소스위치, man 페이지 파일의 위치를 알려주는 명령어
`man [명령어]` : 리눅스의 각종 명령어, 프로그램의 사용법 확인
`ping -c [NUMBER]` : num만큼 패킷을 보냄
`ifconfig` : 현재 네트워크 인터페이스를 설정하거나 확인(내 컴퓨터 주소 정보 알려주는 명령어)
